How to use Harve

Use the dashboard to start sessions, control recording from the pill overlay, and submit when you’re done. Replace the images below with your own product screenshots anytime.

Updated March 2026

1. Starting a session

From the main Harve window, start a recording session when you’re ready to work. Harve talks to our servers to open a session and, when enabled, prepares screen capture. You’ll see the pill — a small floating control — while you’re recording.

  1. Open Harve and make sure you’re signed in.
  2. From the dashboard, choose the action to start recording (wording may match your build).
  3. Grant screen capture permission if the OS asks — required for sessions.

2. The pill overlay

The pill stays on top so you can pause, resume, or finish without digging through menus. It’s designed to be lightweight and out of the way.

3. Pause and sensitive moments

Use Pause when you step away or when you’re in a context you don’t want captured (e.g. a video call). You can also configure automatic pauses for things like banking apps in settings.

4. Submitting a session

When you’re done, submit the session from the pill. Harve finalizes the recording, uploads if your build includes cloud upload, and updates your session history. You’ll see confirmation in the app.

5. Settings and shortcuts

Open Settings from your profile to change theme, keyboard shortcut, auto-pause rules, and more. The shortcut is set during onboarding; you can change it later if the app supports it.
